Key Formulas — Ch 1

Dot Product: \(\mathbf{u}\cdot\mathbf{v}=\sum u_i v_i\)  |  Norm: \(\|\mathbf{v}\|=\sqrt{\mathbf{v}\cdot\mathbf{v}}\)  |  Angle: \(\cos\theta=\frac{\mathbf{u}\cdot\mathbf{v}}{\|\mathbf{u}\|\|\mathbf{v}\|}\)
Line: \(\mathbf{x}(t)=\mathbf{p}+t\mathbf{v}\)  |  Plane (Vector): \(\mathbf{x}=\mathbf{p}+s\mathbf{u}+t\mathbf{v}\)  |  Plane (Normal): \(\mathbf{n}\cdot(\mathbf{x}-\mathbf{p})=0\)
Properties: \(|\mathbf{u}\cdot\mathbf{v}| \le \|\mathbf{u}\|\|\mathbf{v}\|\) (Cauchy-S)  |  \(\|\mathbf{u}+\mathbf{v}\| \le \|\mathbf{u}\|+\|\mathbf{v}\|\) (Triangle Ineq)
Distance: \(d(\mathbf{u},\mathbf{v})=\|\mathbf{u}-\mathbf{v}\|\)  |  Triangle Dist: \(d(\mathbf{u},\mathbf{v}) \le d(\mathbf{u},\mathbf{w}) + d(\mathbf{w},\mathbf{v})\)

Mastering the Method: Vectors & Lines

Find vector of magnitude 7 in opposite direction of v: (1) Normalize \(\mathbf{u}=\mathbf{v}/\|\mathbf{v}\|\). (2) Negate and scale: \(\mathbf{w}=7(-\mathbf{u})\).
Find 3rd vector in Orthonormal Set: Given \(\mathbf{w}_1, \mathbf{w}_2\), find \(\mathbf{w}_3\) by solving \(\mathbf{w}_3\cdot\mathbf{w}_1=0\) and \(\mathbf{w}_3\cdot\mathbf{w}_2=0\). Normalize result. (NO cross product!)
Angle between planes: Use normal vectors \(\mathbf{n}_1, \mathbf{n}_2\). \(\cos\theta = \frac{|\mathbf{n}_1\cdot\mathbf{n}_2|}{\|\mathbf{n}_1\|\|\mathbf{n}_2\|}\).
Plane containing Point P and Line L: (1) Get point \(Q\) and direction \(\mathbf{d}\) from line. (2) Vector \(\mathbf{v}=PQ\). (3) Find normal \(\mathbf{n}\) perp to \(\mathbf{d}\) and \(\mathbf{v}\) using 2x3 system. (4) Use \(\mathbf{n}\cdot(\mathbf{x}-P)=0\).
Closest Point on Plane to Point Q: (1) Line through Q with normal \(\mathbf{n}\): \(\mathbf{x}(t)=\mathbf{Q}+t\mathbf{n}\). (2) Intersection point of this line with plane is the closest point. (3) Distance is \(|t|\|\mathbf{n}\|\).

Key Theorems — Ch 2

Nature of Solutions: Exactly one, Infinitely many, or No solution (Inconsistent).
Rank-Nullity: \(\text{rank}(A)+\text{nullity}(A)=n\) (#cols)  |  Nullity: # of free variables.
Consistency: \(Ax=\mathbf{b}\) is consistent \(\Leftrightarrow \text{rank}(A)=\text{rank}([A|\mathbf{b}])\).
Homogeneous: \(Ax=\mathbf{0}\) always consistent (\(\mathbf{x}=\mathbf{0}\)). Non-trivial sol iff \(\text{rank}(A) < n\).

Mastering the Method: Systems Mastery

Determine Unique/No/Many Solutions: Row reduce [A|b] to REF. (1) No solution: row [0 0 ... 0 | c] where c≠0. (2) Unique: every column has a pivot (no free variables). (3) Many: at least one free variable.
Vector Form Solution: Express basic variables in terms of free variables (parameters \(s, t\)). Write \(\mathbf{x} = \mathbf{x}_p + s\mathbf{v}_1 + t\mathbf{v}_2\).
Testing Linear Independence: Set up \(Ac=\mathbf{0}\). Independent iff only trivial solution (\(\text{rank}=k\)). Dependent iff rank < k (free vars exist).
Maximal Independent Subset: Place vectors as columns of A, row reduce. Pivot columns of ORIGINAL A form the largest independent set.
Polynomial Interpolation (Ch 2 App): To find cubic \(ax^3+bx^2+cx+d=y\) through points, set up a system of equations by plugging in each \((x,y)\). Solve the resulting augmented matrix for \([a,b,c,d]\).
Parameter Analysis (a, b): Row reduce until you have a row like \([0 \dots (\text{expr1}) | (\text{expr2})]\). (1) No sol: expr1=0 and expr2≠0. (2) Many sol: expr1=0 and expr2=0. (3) Unique: expr1≠0.
